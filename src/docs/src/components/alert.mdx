# Alert

Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.
The Alert is a combination of `.box` and color classes.
<p>To see all the colors and their variations go to the <a className='pink' href='/styleguide'>Styleguide page</a></p>

<div class='box background-green white'>
  <p
    class='white'
  >
    <i class="fas fa-check mr-2"></i> I`m a good Alert!
  </p>
</div>
<div class='box background-red white'>
  <p
    class='white'
  >
    <i class="fas fa-times mr-2"></i> I`m a bad Alert!
  </p>
</div>
<div class='box background-blue white'>
  <p
    class='white'
  >
    <i class="fas fa-info-circle mr-2"></i> I`m a info Alert!
  </p>
</div>
