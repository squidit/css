# Button

Button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

<p>
  To see all the colors and their variations go to the{" "}
  <a className="pink" href="/styleguide">
    Styleguide page
  </a>
</p>

## Default

To change color just use `button-:color` class, like `button-pink` or `button-lilac`

<button class="button button-pink">Button</button>

```html
<button class="button button-pink">
  Button
</button>
```

## Rounded

Style with border rounded, just add the class `.rounded`

<button class="button button-pink rounded">Button</button>

```html
<button class="button button-pink rounded">
  Button
</button>
```

## Inverted

Just add `inverted` class

<button class="button inverted button-pink">Button</button>

```html
<button class="button inverted button-pink">
  Button
</button>
```

## Disabled

Just add `disabled` class

<button class="button button-pink disabled">Button</button>

```html
<button class="button button-pink disabled">
  Button
</button>
```

## Sizes

Buttons has sizes class

```css
.button-sm
.button-md
.button-lg
.button-xl
```
