# Color Tokens

Color tokens are the main source of information for naming and cataloging design decisions and front-end alignment regarding colors.
## Roles
Color roles describe the intention behind the color. <br/>
Aligned with the concepts of color psychology, these dads work to reduce the user's cognitive load when identifying or carrying out an action.
