# Color Tokens

Color tokens são a principal fonte de informação para a nomeação e catalogação das decisões de design e alinhamento com front-end referente as cores.

## Roles

Os papéis das cores, ou colors roles, descrevem a intenção por trás da cor. <br/>
Alinhadas aos conceitos de psicologia das cores, esses papais atuam para reduzir a carga cognitiva do usuário na hora de identificar ou realizar uma ação.
